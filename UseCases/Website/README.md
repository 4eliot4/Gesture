# ASL Hub (HAND & SIGNIT)

A simple Next.js hub for model downloads and use-case pages.

## Run Locally

```bash
npm install
npm run dev
```

Visit http://localhost:3000.

## Customize
- Update links on `/downloads` with real model URLs.
- Embed your HAND webcam pipeline and SIGNIT quiz widgets.
- Adjust colors in `tailwind.config.ts` and `app/globals.css`.
