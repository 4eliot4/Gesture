# GESTURE

This repo contains the code for the GESTURE 2025-2026 edition. It contains: 

- V1-2 Folder: Folder containing any and all relevant code from V1-2
  - TGCN: Basis for GESTURE Lite. It contains code for the TGCN edition of the code from GESTURE V2
  - UniSign: Basis for GESTURE Heavy. Empty for now. It will contain code for the UniSign edition of the code from GESTURE V2 when it becomes relevant

- GESTURE Folder: Folder containing any and all relevant code from Fall Semester onwards
  - GESTURE Lite: code for GESTURE Lite.
  - GESTURE Heavy: code for GESTURE Heavy.


- Use Cases Folder: Folder containing anything related to the GESTURE product, that is the code for the use cases and the website
  - SIGNIT: code for SIGNIT
  - HAND: code for HAND
  - Website: code for the website.
